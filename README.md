## üß† Service Contract Scheduling Algorithm  
A DAX-driven logic engine automatically assigns each product into:

| Service Year | Logic |
|--------------|-------|
| **Y1** | Purchase date ‚Üí end of year 1 |
| **Y2** | Follows Y1 based on contract rules |
| **Y3** | Follows Y2 automatically |

The algorithm dynamically recalculates based on:

- Purchase date  
- Manufacturing date  
- Warranty length  
- Lead time  
- Contract duration  
- Product type  

This eliminated manual categorisation and improved forecasting accuracy.
